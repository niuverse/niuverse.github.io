# Niuverse

ruziniuuuuu 的个人项目网站

🔗 在线访问: https://niuverse.github.io

## 关于

Niuverse 是我的个人项目展示空间，用于记录创意和分享作品。

## 技术栈

- HTML5
- CSS3
- GitHub Pages

## 本地预览

```bash
# 克隆仓库
git clone https://github.com/niuverse/niuverse.github.io.git
cd niuverse.github.io

# 用浏览器打开
open index.html
```

## 自定义域名

配置自定义域名 niuverse.io：
1. 在仓库 Settings > Pages 中添加自定义域名
2. 在域名 DNS 中添加 A 记录指向 GitHub Pages IP
3. 等待 DNS 生效
